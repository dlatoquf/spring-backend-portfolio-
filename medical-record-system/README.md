# 🏥 Medical Record System
> 환자 정보 및 진료 기록을 관리하는 의료 데이터 시스템

---

## 🚀 핵심 성과
- 환자 정보 및 진료 기록 CRUD 기능 구현
- 데이터 구조 설계 및 관리 경험 확보
- 의료 데이터 흐름 이해 및 시스템 구현 경험

---

## 📌 프로젝트 소개
환자 정보 및 진료 기록을 관리하기 위한 의료 데이터 시스템입니다.  

환자의 기본 정보부터 진료 기록까지 관리할 수 있도록  
데이터 구조를 설계하고 CRUD 기능을 구현했습니다.

---

## 🛠 기술 스택
- **Backend**: Node.js (Express)  
- **Frontend**: HTML, CSS, JavaScript  
- **Database**: MySQL  
- **Tools**: Git  

---

## ⚙️ 주요 기능
- 환자 정보 등록 / 수정 / 조회
- 진료 기록 관리
- 환자 데이터 검색 기능

---

## 🔥 핵심 구현

### 1. 데이터 구조 설계
- 환자 정보 및 진료 기록 데이터 구조 설계
- 관계형 데이터 기반 CRUD 기능 구현

---

### 2. 환자 데이터 관리 기능
- 환자 등록 및 수정 기능 구현
- 진료 기록 저장 및 조회 기능 구현

---

### 3. 서버 로직 구현
- Node.js 기반 서버 구성
- API 요청 처리 및 데이터 반환

---

## 💥 트러블 슈팅

### 문제
환자 데이터와 진료 기록 간 관계 설정 과정에서 데이터 관리의 어려움 발생

### 해결
데이터 구조를 재설계하여 관계를 명확히 구성

### 결과
데이터 조회 및 관리 효율 향상

---

## 📊 데이터 흐름
1. 환자 정보 입력
2. DB 저장
3. 진료 기록 추가
4. 조회 요청 시 데이터 반환

---

## 👨‍💻 개발 회고
이 프로젝트를 통해 데이터 구조 설계의 중요성을 이해할 수 있었으며,
