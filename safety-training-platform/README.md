# 🦺 Safety Training Platform
> Unity 기반 메타버스 안전 교육 플랫폼

---

## 🚀 핵심 성과
- Unity 기반 3D 환경에서 안전 교육 콘텐츠 구현
- 사용자 상호작용 기능 개발
- 팀 프로젝트 협업 경험

---

## 📌 프로젝트 소개
메타버스 환경에서 안전 교육을 제공하기 위한 플랫폼입니다.  

사용자가 직접 참여하며 학습할 수 있도록  
3D 환경 기반 인터랙션 기능을 구현했습니다.

---

## 🛠 기술 스택
- **Engine**: Unity  
- **Language**: C#  
- **Tools**: Visual Studio  

---

## ⚙️ 주요 기능
- 안전 교육 시나리오 구현
- 사용자 이동 및 상호작용 기능
- 이벤트 기반 학습 콘텐츠 제공

---

## 🔥 핵심 구현

### 1. 사용자 인터랙션 구현
- 버튼 및 이벤트 기반 상호작용 기능 구현
- 사용자 행동에 따른 이벤트 처리

---

### 2. 교육 콘텐츠 구성
- 안전 교육 시나리오 기반 기능 구현
- 단계별 학습 흐름 구성

---

## 💥 트러블 슈팅

### 문제
사용자 이벤트 처리 과정에서 기능 충돌 발생

### 해결
이벤트 처리 로직을 분리하여 구조 개선

### 결과
안정적인 사용자 인터랙션 구현

---

## 👨‍💻 개발 회고
이 프로젝트를 통해 팀 협업 경험을 쌓을 수 있었으며,  

사용자 경험을 고려한 기능 구현의 중요성을 이해할 수 있었습니다.
