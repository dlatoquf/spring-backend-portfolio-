# 🦺 Safety Training Platform (STS)
> Unity 기반 메타버스 안전 교육 플랫폼

---

## 📌 프로젝트 요약

### 📌 프로젝트 개요
* 프로젝트명: Safety Training Platform (STS)
* 개발 기간: 2023.09 ~ 2023.11 (약 3개월)
* 개발 인원: 5명 (팀 프로젝트)
* 역할: 기능 구현 (Raycast, UI), 이벤트 로직 설계, 일부 화면 구현
* 플랫폼: Unity 기반 메타버스

---

### 🛠 개발 환경
* Engine: Unity  
* Language: C#  
* Tools: Visual Studio  

---

### 🚀 핵심 성과
- 3D 메타버스 환경에서 체험형 안전 교육 시스템 구현  
- Raycast 기반 객체 상호작용 기능 개발  
- 이벤트 기반 학습 흐름 설계 경험  
- 팀 프로젝트 협업 및 역할 분담 경험  

---

## 📌 프로젝트 소개
기존 안전 교육은 이론 중심으로 진행되어  
실제 상황 대응 능력을 키우기 어렵다는 한계가 있습니다.  

이를 개선하기 위해  
사용자가 직접 참여하며 학습할 수 있는  
**메타버스 기반 안전 교육 플랫폼(STS)**을 개발했습니다.

---

## 🎯 문제 정의

![문제 정의](./images/sts-problem.png)

- 이론 중심 교육으로 실전 대응 능력 부족  
- 실제 상황을 체험할 수 있는 환경 부족  
- 학습 몰입도 저하  

👉 이를 해결하기 위해  
**체험형 3D 안전 교육 시스템 설계 및 구현**

---

## ⚙️ 주요 기능
- CPR 및 화재 상황 시나리오 구현  
- 사용자 이동 및 상호작용 기능  
- 이벤트 기반 학습 진행 구조  
- 텍스트 안내 UI 시스템  

---

## 🔥 핵심 구현

### 1. 텍스트 안내 시스템

![텍스트 안내](./images/sts-text-ui.png)

- 사용자 행동을 유도하는 안내 메시지 출력  
- 단계별 학습 흐름 제어  
- 버튼 입력 기반 이벤트 진행 구조 구현  

---

### 2. Raycast 기반 상호작용

![Raycast](./images/sts-raycast.png)

- Raycast를 활용한 객체 감지 기능 구현  
- 특정 오브젝트와 상호작용 시 이벤트 발생  
- 사용자 행동에 따른 분기 처리 로직 설계  

---

### 3. 시나리오 기반 이벤트 처리

![시나리오](./images/sts-scenario.png)

- CPR 및 화재 상황 시나리오 구성  
- 단계별 이벤트 흐름 제어  
- 사용자 입력 기반 상황 진행 로직 구현  

---

## 💥 트러블 슈팅

### 문제  
사용자 이벤트 처리 과정에서  
이벤트 충돌 및 흐름 꼬임 발생  

### 해결  
- 이벤트 로직을 단계별로 분리  
- 상태값 기반 흐름 제어 구조 적용  

### 결과  
안정적인 이벤트 처리 구조 확보 및  
사용자 경험 개선
