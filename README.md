# 실무형 백엔드 개발 포트폴리오

> Java/Spring 기반 REST API 설계, 데이터 처리, 배포·운영까지 전 과정을 경험한 백엔드 개발자입니다.

---

## About Me

안정적인 서비스 운영과 성능 개선에 강점을 가진 백엔드 개발자입니다.

실무에서 1인 개발자로 요구사항 분석부터 DB 설계, API 구현, Linux 서버 배포 및 운영까지 전 과정을 주도하였습니다.
데이터 정합성 검증, SQL 최적화, 트랜잭션 처리 등 서비스 신뢰성을 높이는 개발에 집중해왔으며,
운영 중 발생하는 장애를 로그 분석을 통해 근본 원인을 파악하고 해결하는 경험을 쌓아왔습니다.

## 프로젝트

### 1. SMS Payment Service

> 실무 프로젝트 — 제휴·결제 시스템 개발 및 운영

**개요**
가맹점·영업점 기반 SMS 결제 처리 시스템의 백엔드 개발 및 운영을 1인으로 전담하였습니다.

**주요 구현 내용**
- RESTful API 기반 결제 요청·승인·취소 프로세스 설계 및 구현
- 중복 요청 방지 및 예외 처리 로직 적용으로 데이터 정합성 확보
- 사용자·가맹점·영업점 등록 및 조회 기능 개발
- Excel 대량 업로드 기능 및 입력값 유효성 검증 로직 구현
- SQL 최적화를 통한 대량 데이터 조회 성능 개선
- Linux 환경 배포 및 운영 중 장애 대응

**기술 스택**
Java, Spring Framework, MyBatis, MariaDB, JavaScript, jQuery, Linux

### 3. Medical Record System

> 환자 정보 및 진료 기록 관리 시스템 (대학교 졸업 프로젝트)

**개요**
의료IT공학 전공 프로젝트로, HL7 FHIR 표준을 활용한 웹 기반 EMR 시스템을 개발하였습니다.
PM 및 백엔드 개발을 담당하였으며, 서비스 기획부터 데이터 흐름 설계까지 주도하였습니다.

**주요 구현 내용**
- Node.js 기반 서버 구축 및 REST API 개발
- HL7 FHIR 표준을 활용한 환자·진료 데이터 송수신 로직 구현
- 진료 상태(대기/진행/완료)에 따른 실시간 데이터 변경 기능 개발
- 환자 상태를 직관적으로 파악할 수 있는 간호사 화면 UI 개발

**기술 스택**
Node.js, JavaScript, MySQL, HTML, CSS, HL7 FHIR

- 팀 프로젝트 PM으로 기획·시나리오 설계 참여

**기술 스택**
Unity, C#
