# 🧑‍💻 Spring Backend Portfolio
> 대용량 데이터 처리 및 운영 경험을 갖춘 실전형 백엔드 개발자

---

## 👤 About Me

안정적인 서비스 운영과 성능 개선에 강점을 가진 백엔드 개발자입니다.  
Java/Spring 기반 API 개발과 데이터 정합성 검증, SQL 최적화를 통해 서비스 품질을 개선해왔습니다.  
특히 데이터 처리 흐름과 트랜잭션을 고려한 로직 설계를 통해 서비스 신뢰성을 높이는 데 집중해왔습니다.

기획부터 DB 설계, API 구현, 배포 및 운영까지 전 과정을 경험하며  
실제 서비스 환경에서 발생하는 문제를 해결하는 개발자로 성장하고 있습니다.
